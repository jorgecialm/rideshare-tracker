# Rideshare Tracker

Seguimiento de jornadas y métricas para conductores de Uber.

## Funcionalidades

- Registro diario de jornadas (tiempo conectado, viajes, monto)
- Métricas automáticas: ganancia por hora y por viaje
- Comparativa por día de la semana
- Totales y promedios semanales, mensuales y anuales

## Stack

- Python 3 / Django
- SQLite (desarrollo)
- Templates responsive
